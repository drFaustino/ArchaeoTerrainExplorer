description=ArchaeoTerrain Explorer is an interactive micro‑topographic analysis tool designed for archaeological prospection. The plugin extracts a 256×256 DEM window around the clicked point and computes real‑time terrain metrics including slope, curvature (profile, plan, total), Local Relief Model (LRM), hillshade, openness (positive/negative), Sky‑View Factor (SVF), and Multi‑Scale Relief Model (MSRM). It offers both composite and single‑filter visualization modes (Slope, Curvature, LRM, Hillshade, Openness+, Openness‑, SVF, MSRM), along with archaeological presets such as Archaeo‑Enhance 1 (LRM + openness), Archaeo‑Enhance 2 (SVF + curvature), Anthropic Edge Detector, and Paleochannel Finder. Features include scan‑mode preview, a recompute‑preview button for the same point when changing filters, customizable parameters, colormap selection, a dynamic progress bar, and export to GeoTIFF or PNG.
